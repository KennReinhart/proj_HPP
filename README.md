# House Price Prediction Using Linear Regression

## Objective
Predict house prices based on property characteristics to support real estate pricing decisions.

## Dataset
Housing data including size, room count, location score, and age.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Methodology
- Exploratory data analysis
- Linear regression modeling
- Train-test evaluation
- Error metric interpretation

## Results
- Mean Absolute Error (MAE): ~25,000
- Root Mean Squared Error (RMSE): ~30,000
- Strong influence from size and location quality

## Business Use
- Price estimation
- Negotiation support
- Market benchmarking

## Limitations
- Limited dataset
- Linear assumptions
- No external market factors

## Next Steps
- Larger datasets
- Advanced models
- Feature expansion
